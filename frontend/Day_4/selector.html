Selector

Universal Selector

it indicates with *

it select all elements

Grouping Selector

it is select group of elements
it is indicated with comma(,)

Combine Selector

    descendent selector
    it select all child element(direct/indirect)

                parent  child{
                //both direct and indirect child will select

            }
    child selector

    it select direct child element

                parent > child{
                //it select direct child
            }
    adjacent sibling selector

    it select immediate adjacent element

                main + adjacentelement{
                //it select immediate adjacent element
            }
    General sibling selector

    it select all adjacent element

                main ~ adjacentelement{
                //it select all sibling to the main element

            }
