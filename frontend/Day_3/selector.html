selector

selector is used to select the element in the html

selector is used to apply css to the selected element

selector is used to apply js to the selected element

types of selector

  1. element selector
  2. id selector
  3. class selector
  4. attribute selector
  5. universal selector
  6. group selector
  7. descendant selector
  8. child selector
  9. adjacent sibling selector
  10. general sibling selector

element sector

element selector is used to select the element based on the tag name

class selector

  • class selector is used to select the element based on the class name
  • class selector not be unique and can select multiple element
  • we can use class selector with .className for css style

id selector

  • id selector is used to select element based on id names
  • id selector must be unique
  • we can use id seletor with #idName for css style

Rules:

  1. class/id name should have alphabetes and no (- , _)
  2. class/id name not to start with no
  3. className and idName can be same
